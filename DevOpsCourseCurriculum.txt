DevOps Course Curriculum
====================
Tools Covered:
Linux and Shell Scripting - prerequisites
Git and Github - VCS and SCM
Jenkins - CICD
Docker - containers
Kubernetes - container orchestration
Ansible - Config Management
Terraform - IAC
AWS -  Cloud
Prometheus - monitoring servers, apps, and services
Grafana - Data visualization
Jira - Issue tracking tool
SonarQube - Code Quality Check
